Dataset Description

Dataset Source:
The dataset used in this project is the Titanic dataset, which is available on Kaggle. It contains information about passengers who were on the Titanic ship, including their personal information and whether they survived or not.

Number of Rows and Columns:
The dataset contains 891 rows and 12 columns.

Description of Features (Columns):
-PassengerId: Unique ID for each passenger.
-Survived: Survival status (0 = Did not survive, 1 = Survived).
-Pclass: Passenger class (1 = First class, 2 = Second class, 3 = Third class).
-Name: Name of the passenger.
-Sex: Gender of the passenger.
-Age: Age of the passenger.
-SibSp: Number of siblings/spouses aboard the Titanic.
-Parch: Number of parents/children aboard the Titanic.
-Ticket: Ticket number.
-Fare: Ticket price paid by the passenger.
-Cabin: Cabin number.
-Embarked: Port of embarkation (C = Cherbourg, Q = Queenstown, S = Southampton).

Purpose of Using This Dataset:
The purpose of using this dataset is to perform Exploratory Data Analysis (EDA), including data cleaning, data visualization, and finding patterns and relationships between different features that may affect passenger survival.
